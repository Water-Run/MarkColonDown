# 语法糖  

语法糖是`Mark:down`的最简单的使用方式, 只是`Mark:down`原生语法的另一种表示.  

## 标题  

标题语法糖用于省略标题的字符输入, 并使得层级更加直观.  

只需要:  

```markcolondown
#:[对应层级的数字, 1-6]
```

例如:  

```markcolondown
#:3 三级标题
等价于
### 三级标题
```

## 链接行和图片行  

链接行和图片行用于简化一个表达一个链接或图片的行.  

使用`#:!`和`#:&`作为语法:  

```markcolondown
#:! ./我的图片路径.png
等价于
![](./我的图片路径.png)  

如果需要图片注解(即[]部分), 在路径的结尾要加入():  

#:! ./我的图片路径.png(图片的解释信息)
等价于
![图片的解释信息](./我的图片路径.png)  

对于链接行, 语法也相同.  

#:& https://github.com/Water-Run/MarkColonDown
#:& https://github.com/Water-Run/MarkColonDown(Mark:down)
等价
[](https://github.com/Water-Run/MarkColonDown)  
[Mark:down](https://github.com/Water-Run/MarkColonDown)  
特别的, 链接行可以指定前缀. 在`&`之后的空格之前加入即可(不可包含空格):  

#:&- https://github.com/Water-Run/MarkColonDown(Mark:down)
编译为
- [Mark:down](https://github.com/Water-Run/MarkColonDown)  
或
#:&GitHub: https://github.com/Water-Run/MarkColonDown(Mark:down)
GitHub: [Mark:down](https://github.com/Water-Run/MarkColonDown)  
```

> 也可以使用全称的形式: `#:image`和`#:link`

> 行结尾的两个空格会自动添加使换行

## 表格  

`Markdown`原生的表格语法也略显冗余, 且在希望新增/删除列时很麻烦.  
`Mark:down`提供了表格便携语法: `#:-`表示一个表格块, 到`#:=`内遵循CSV的语法:  

```markcolondown
#:-
列1, 列2, "带,的列"
值1, 值2, 值3
123, "123", "1,2,3"
#:=
```

> 同样的, 也可以使用全称形式`#:table`
